# Data_Warehouse_project
Objective Develop a modern data warehouse using SQL server to consolidate sales data, enabling analytical reporting and informed decision making and also develop SQL based analytics to deliver insight into Customers behaviour, Product performance and Sales trends.
Specifications
Data Source→ Import data from two systems ERP and CRM as csv files.
Data Quality→ Cleanse and resolved the data quality issues.
Integration→ Combine both sources into single; user-friendly data model
Scope→ Latest dataset only: historization is not required.
